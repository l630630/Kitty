组件库文档

欢迎使用我们的组件库！这个库提供了一系列可复用的React组件，帮助你快速构建现代化的用户界面。

安装

使用npm或yarn安装组件库：

# 使用npm
npm install @my-component-library/ui-core

# 使用yarn
yarn add @my-component-library/ui-core

# 使用pnpm
pnpm add @my-component-library/ui-core

快速开始

import { Button } from '@my-component-library/ui-core';

function App() {
  return (
    <div>
      <h1>Hello World</h1>
      <Button>点击我</Button>
    </div>
  );
}

可用组件

  • Button 按钮 - 用于触发操作的交互元素
  • Drawer 抽屉 - 从屏幕边缘滑出的浮层面板
