# 组件库文档

欢迎使用我们的组件库！这个库提供了一系列可复用的React组件，帮助你快速构建现代化的用户界面。

## 安装

使用npm或yarn安装组件库：

```bash
# 使用npm
npm install kitty-react

# 使用yarn
yarn add kitty-react

# 使用pnpm
pnpm add kitty-react
```

## 快速开始

```jsx
import { Button } from 'kitty-react';

function App() {
  return (
    <div>
      <h1>Hello World</h1>
      <Button>点击我</Button>
    </div>
  );
}
```
